Навчальні роботи

Проєкти та практичні завдання

Нижче — приклади напрямів, у яких я зараз розвиваюся. Сітка карток побудована на CSS Grid з `auto-fit` та `minmax()`, тому автоматично адаптується до ширини екрана.

HTML5

Багатосторінкове портфоліо

Семантична структура сайту з окремими сторінками, навігацією, таблицями, формами та медіа.

Фокус: семантика, логічна структура, доступність.

CSS Grid

Адаптивний макет

Побудова сторінки з областями header, content, sidebar і footer через grid-template-areas.

Фокус: mobile first, брейкпоінти, сучасна верстка.

Flexbox

Гнучка навігація

Адаптивне меню з гамбургером на мобільних пристроях та горизонтальним розташуванням на планшеті й десктопі.

Фокус: align-items, justify-content, gap.
